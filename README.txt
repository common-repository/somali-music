Somali Music Player lets you stream somali  music from masuul or maalmo, it shows on footer with nice music player.


== Description ==
Somali music player in any website
<br>
<br>
Somali  Music Player is a free  plugin you can add your web page.
<br>
Just, download, install, and activate Somali Music . then you will see on your footer somali music player  with browser notifications.

= Features =
* Stream tracks from Masuul or maalmo.
* Play, pause, skip and shuffle controls, remove any songs on playlist for users.
* display notifications to the user currect playling song with you website name.
* Fixed footer .
* Mobile responsive on any device.
* you can choose any somali song you like  from qaraami to new songs.
<br>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `Somali music` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. if you want to chanche the musics you can change in admin panel somali music
